# Dreamy Localization

CSV-first localization package for Unity 6 and DreamyBase projects.

The package uses Unity Localization for locales, String Tables, Asset Tables,
Smart Strings, and Addressables integration.

## Status

Version `0.1.0` contains the package foundation:

- Runtime localization service and typed keys.
- Locale persistence and culture alias matching.
- TMP and UGUI string bindings.
- CSV parser, writer, schema model, and validation.
- EditMode tests for the CSV domain.

See [LOCALIZATION_PACKAGE_PLAN.md](LOCALIZATION_PACKAGE_PLAN.md) for the full
implementation roadmap.
